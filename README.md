# 📚 Library Management System (Java OOP Project)

## ✅ Objective
A mini **Library Management System** developed in **Java** to manage books and users using core **Object-Oriented Programming (OOP)** concepts.



---

## 📂 Project Structure
- `Book.java` – Book class with properties and methods
- `User.java` – Represents library users
- `Library.java` – Manages books, users, issue/return logic
- `Main.java` – Application entry point with CLI menu

---

## 🎯 Features
- Add and view books and users
- Issue a book to a user
- Return a book
- OOP principles used:
  - **Encapsulation**
  - **Abstraction**
  - **Inheritance**
  - **Polymorphism**
